# piglin-barter-distro
CLI for calculating probability distributions for piglin bartering in Minecraft. Results will only be correct for Minecraft versions 1.16.2 - 1.21.5, piglin barter chances are different outside of this range.

### Running
1. Clone this repo
2. `cd` into the folder, and run `cargo run --release`

You will need Rust and Cargo installed. I recommend using [Rustup](https://rustup.rs/).
